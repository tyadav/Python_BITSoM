Lectures 1–3 Summary
Lecture 1 — Setup, Syntax, and Strings
- Setup & environment; running scripts; IDE workflow
- Variables and basic types: int, float, bool, str
- Numbers & arithmetic: +, -, *, /, //, %, **
- Strings: creation, concatenation, multiline strings, slicing/indexing, reversing
- String methods: upper/lower/title/swapcase, strip/replace, split/join
- String search/inspect: find, count, startswith/endswith, isalnum/isalpha/isdigit,
  partition, splitlines, zfill
- Lists (intro): create, length, indexing/slicing, append/insert/remove/pop
Lecture 2 — Booleans, Conditionals, Loops, Lists
- Boolean expressions: comparisons (==, !=, <, >, <=, >=), logical (and, or, not)
- Bitwise examples and formatting (binary f-strings)
- Conditionals: if, if–else, if–elif–else
- range() and iteration patterns
- Loops: for, while, nested loops; break and continue
- Lists (basics continued): types, slicing, mutation, append/insert/remove/pop
Lecture 3 — Lists (Advanced), Dictionaries, Exceptions
- Lists (deeper): append/extend/insert/remove/pop/clear, index/count, membership (in)
- Sorting: list.sort(), reverse(), sorted() on lists/strings
- Copying semantics: assignment vs copy; shallow vs deep copy (copy.deepcopy), slicing copy
- Dictionaries: literals, dict(), fromkeys; access via []/get(default); update/setdefault
- Dict views & ops: keys/values/items, pop/popitem/clear, membership, len, iteration
- Nested structures: dicts with nested dicts/lists and updates
- Error handling: try/except (ZeroDivisionError, ValueError, IndexError, KeyError,
  TypeError, AttributeError, FileNotFoundError), multiple excepts, finally, custom Exception
